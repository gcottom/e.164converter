Gage Cottom- E.164 Converter
Language: Java

Functionality: My convert function takes a string input (which is retrieved from the keyboard
in the main function), converts it to E.164 format, and returns it back to the main function.
Error checking is performed in the convert function. The main function has a quit option (0). 

How to use: Open the program using java in a command line environment. Type your input and 
output will be returned to the main function and printed to the screen. 

Assumptions: Input is expected in any of 3 formats: 555-555-5555, (555)555-5555, and 5555555555
E.164 format is +15555555555
